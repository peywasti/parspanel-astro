# parspanel-astro

This repository contains the new **ParsPanel website**, rebuilt using [Astro](https://astro.build).  
The goal is to create a fast, lightweight, and SEO-friendly static website to replace the old WordPress version of [parspanel.com](https://parspanel.com).
